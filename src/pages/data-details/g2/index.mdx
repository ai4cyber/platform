## Who generates the data?

The organization 'G2' provides authentic reviews about thousands of medical software products.

***

## Is the data relevant?

It helps to identify the different medical software applications which are used by the hospitals.

***

## Is the data of high quality?

Many organizations trust the reviews of G2 to identify the software they need. G2 repositories are up-to-date.

***

## Is the data accessible?

The data is publicly accessible, and we are extracting it using a web scraping tool which captures the content of the page in a structured way.

***

## Why did we select this data source and what value does it bring?

This data source provides trustful medical software product names, which would be used to identify the different vulnerabilities.

***

## What is the Collection Strategy?

- Identify the information available from G2 web pages.
- Extract the data using a web scraping tool ‘Octoparse'.
- Results are extracted & stored in mySQL Server.
